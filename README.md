# knu_helper

*주요기능*
1. 각 학과별 공지사항을 리스트를 통해 확인할 수 있어요.
2. 새로 등록된 공지사항을 알림을 통해 받아보세요.
3. 저장하고 싶은 공지사항을 즐겨찾기 기능을 통해 남겨두세요.
